# project_web_frameworks
This repository consist of a simple request response cycle by utilizing django framework and it also consist of a sample streamlit application to analyze a specifically formatted bank statement from the UAE.

# Django Application
This application is a very simple level coding practice for beginners and it is used to provide an html structure as a response. Note that the files are initialized congruent to real django project but databases and settings are left off for now.

# Streamlit Applcation
The streamlit application here is named as automate.py and given dataset sample_bank_statement.csv is a dummy data downloaded from the internet for practice. This streamlit app is integrated with pandas library for a simple analysis of the bank statement composed of different credits and debits carried on from th bank and a summary metrics too. Though it is not finished it was a perfect practice to have a gist about time-series analysis, grouping based on specific condition and creating simple dashboards using streamlit.
